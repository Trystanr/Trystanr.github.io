Basics

Let's look at a barebones HTML file:

index.html 
<!DOCTYPE html>
	<head>
		<meta charset="utf-8">
		<title>Title</title>
	</head>
	<body>
		<p>hello world</p>
	</body>
</html>

To view this page in your browser, simply save the file as index.html and double click the file.
You should see the following:

Title 
                    
Hello World

Let's look at what we have here.

!DOCTYPE html indicates that this web page is written in html.
head contains the web page title.
meta charset="utf-8" specifies the character encoding for the HTML document.
title contains meta data (data about data) about the page.
body contains the content of the web page.
p is a paragraph element.
